css - 인테리어

JavaScript - 집의 기능


오늘은 셋 중 하나인 HTML의 기본적인 태그들을 배웠다.


HTML(Hypertext Markup Language) - 마크업 언어

마크업 언어란? -> 태그를 이용해서 구조화하는 언어


<head>태그

메타 데이터(데이터에 대한 정보 제공 데이터), 문서 정보 정의에 쓰임

-title : 제목 붙이기

-style : 꾸미기

-link : 링크 연결

-script : javascript 연결

-meta : 메타데이터 정의


<body> 태그

실제 컨텐츠 정의, 웹 페이지 구조 설계


[문법]

요소(Element)

<시작 태그> 내용 </종료 태그> 로 이루어짐


중첩(nested)

들여쓰기를 활용해 가독성 높임

부모-자식 관계

<부모태그>

<자식태그>내용</자식태그>

</부모태그>


빈 요소(Empty)

내용이 없는 요소 but 속성은 있음

ex) <meta charset="utf-8">


속성(Attribute)

<태그 속성="값">내용</태그> 형식

요소의 성질 특성 정의, 추가적 정보(경로,크기 등) 제공


주석(Comments)

코드를 설명하기 위해 사용(코드에 포함 X)

<!-- 주석입니다(하주석 아님) --> 형식


제목 태그<h1~6></h1~6>

1부터 6 순서로 크기가 작아짐


본문 태그<p></p>

paragraph의 약자

말 그대로 본문을 넣는 태그


줄 바꿈 태그<br>

줄을 바꿔준다. break의 약자.


목록, <ul></ul> or <ol></ol>

차이점은 순서의 유무

unorderd list, orderd list의 차이

type을 설정할 수 있다.

ul type = circle(빈 원), square(사각형)

수평 줄,<hr>

수평으로 된 줄을 그어준다.


문자 꾸미기 태그

<b> : 굵게

<i> : 이텔릭

<em> : 문자 비스듬히


이미지 넣어주는 태그<img>

속성 값 중 하나인 src를 사용

<img src = "이미지 링크 or 파일 경로" alt = "이미지가 없을 시 표시할 문구">


하이퍼링크 <a>

Anchor의 약자인 <a>태그 사용

<a href="링크" target="_blank or _self">문구</a>

_blank = 새로운 창에 링크된 사이트 열기

_self = 현재 창을 링크된 사이트로 이동


input태그

<input type="타입 입력">

type의 종류는 다음과 같다.

text - 텍스트 입력 창

password - 텍스트 입력 창과 동일하나 입력한 문자가 가려짐

checkbox - 체크박스(여러 개 가능)

radio - 라디오버튼(하나만 가능)
